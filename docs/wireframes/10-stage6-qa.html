Inspector swarm — agent probe loop

inspector · Qwen3.6-35B-A3B Functional

4 concurrent agents, each a bounded tool loop (≤15 steps, ≤60 s). Tools: http_request (egress-scoped to the staging host 127.0.0.1:<ephemeral>), read_manifest, submit_finding. Contract: you can't see the code — try to break the claim, not fix it; every finding needs a reproducible request/response pair.
