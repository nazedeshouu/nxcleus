streaming · GET /egress + /egress/stream SSE

Every external / boundary-crossing call the scope makes — the wall you can watch. On-node model dispatches (Local) never appear here; only AMD-hosted Fireworks and External providers / 3rd-party data fetches cross the boundary, and each must carry Sanitized data. A RAW→External attempt is refused by the router and raised as egress.violation (Sovereign red banner).

🔧 Tools registry  Functional Stub-fallback

GET /tools?scope= · 404 → tab hidden · live-append via tool.created SSE

Python tools the coder/consolidator seats commissioned at build time — each with a self-test the runtime ran before admitting it. Backed by GET /tools; on a backend without the endpoint the query 404s and the tab is hidden (no fixture list ships, so treat the populated state as stub-fallback demo data).

▸ 🔧 sanctions_match ✓ self-test coder · qwen2.5-72b

Fuzzy-screen a subject name against OFAC SDN + EU CFSP lists; returns best-match score and list source. Expand for args_schema + emitted code.

▸ 🔧 risk_score ✓ self-test coder · qwen2.5-72b

Weighted risk score from PEP flag, adverse-media hits, and jurisdiction tier; deterministic — cross-checked by the oracle seat.

▸ 🔧 edd_packet no self-test consolidator · qwen2.5-72b

Assemble an Enhanced Due Diligence packet (sanitized) for L2 review when risk ≥ threshold. Expand for schema + code.
