Stage 4 — Parallel Codegen in Waves  /build/job_01JZF3KYCAML · stage 4

The live build cockpit. The engine partitions the certified DAG into topological waves; a coder pool builds each wave in parallel — every task routed to the pool member whose capability flags best match its task_flags — and the conductor reviews between waves against the pinned goal before green-flagging the next. Watched by the operator running the KYC/AML onboarding build.

Goal — what we promised FN

Onboard a bank applicant end-to-end: normalize the KYC record, OCR the identity documents, screen against OFAC + EU consolidated sanctions and the PEP list, compute a weighted risk_score (rule NR-1) into amber/red bands, and emit a reviewable case file with an auto-drafted SAR narrative when the band is red. Judged holistically against the customer's original request at stage 6. · jobs.goal, set at certify.goal_set
