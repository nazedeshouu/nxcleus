**AMD Developer Hackathon: ACT II**
Build AI agents and high-performance AI apps on AMD GPUs in the cloud · Prize Pool $20,000+
Online · Monday, July 6 2026 — 9:00 PM Kazakhstan Time
Submission deadline: Jul 11, 9:00 PM KT

## Registration & credits notice
- Register by **July 2nd** to receive hackathon credits on day one.
- Registration stays open after July 2nd, but credits for late sign-ups are allocated **from July 7th onward**.
- **Two separate credit types:**
  - **Hackathon credits** — all participants receive **$50 in Fireworks AI API credits**. AMD Developer Cloud GPU access details shared once the event starts.
  - **New member credits** — **$100 AMD Developer Cloud + $50 Fireworks AI** for new AMD AI Developer Program (ADP) sign-ups. Separate 2–3 day manual approval; not affected by the hackathon cutoff.

## What participants get
- $100 AMD Developer Cloud GPU credits (new ADP sign-ups only)
- $50 Fireworks AI API credits (new ADP sign-ups only)
- AMD AI Academy (courses, docs, tutorials)
- DeepLearning.AI Pro — 1 month complimentary (new sign-ups only)
- AMD Expert Access (engineers, office hours)
- Private community access
- Project recognition (AMD social/events showcase)
- Fully cloud-based (no local hardware)

## Challenge — three tracks
- **Track 1 — Hybrid Token-Efficient Routing Agent** · ⭐ Beginner · AI Agent Track
- **Track 2 — Video Captioning** · 🎬 Beginner · Prompt and have fun
- **Track 3 — Unicorn Track** · 🦄 All levels · Build your startup
- All submissions must be containerized.

## Technology & access
- AMD Developer Cloud (on-demand AMD GPUs), ROCm (open-source GPU compute platform), Fireworks AI API (AMD-hardware hosted models), open-source frameworks. Fully cloud-based.

## Technology partners
- **Gemma** (Google DeepMind) — accessed via Fireworks AI + AMD Developer Cloud; Apache 2.0. "Best Use of Gemma" prizes.
- **Native.Builder** (NativelyAI) — AI-native build environment; use Fireworks AI credits inside Builder.

## Prizes — Total Prize Pool: $21,000
- Each track: 🥇 $2,500 · 🥈 $1,500 · 🥉 $1,000
- Google DeepMind Gemma prizes: **$6,000** — Track 1 $1,000 · Track 2 $3,000 · Track 3 $2,000 (Best AMD-Hosted Gemma Project)
- Referral pool: **$1,000** — top referrer $250 cash + $250 Natively credits; 2nd $150+$150; 3rd $100+$100. Must refer ≥100 approved participants to qualify.
- Submissions must be original and MIT-compliant. Prize distribution may take up to 90 days.

## What to submit (lablab platform)
- Basic info: Project Title, Short Description, Long Description, Technology & Category Tags
- Cover Image & Presentation: Cover Image, Video Presentation, Slide Presentation
- App hosting & code: Public GitHub Repository, Demo Application Platform, Application URL
- Requirements: submit before deadline; all submissions containerized; public GitHub repo with README (setup + usage); app runnable from provided instructions.

## Judging criteria
- Tracks 1 & 2: ranked via leaderboard (see track details).
- Track 3 — Unicorn: judged on Creativity & Originality, Product/Market Potential, Completeness, Use of AMD Platforms.
